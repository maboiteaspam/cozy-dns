# Cozy-dns

A frontend UI to manage a bunch of DNS&DHT able node.

Create a peer.

It will connect to a local DHT table.

Attach it a domain.

Try to resolve the domain from this node or a new you d create.


### Run it

```
mkdir myplayground
cd myplayground
npm i maboiteaspam/cozy-dns
node index.js
```
